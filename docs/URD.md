# メシとも ー ユーザー要求定義書


## サービス名

**メシとも** — 飲み・食べ放題マッチングサービス

## サービス概要

一人飲みが好きだけど、たまには誰かと食べ放題・飲み放題に行きたい人のためのマッチングサービス。見ず知らずの人同士が「予算・食べたいもの・地域」でつながり、完全割り勘で気軽に参加できる。

## ターゲットユーザー

- 一人飲みが好きだけど、たまには誰かと行きたい人
- 20歳以上限定（飲酒を前提とするため）

## 基本仕様

### プラットフォーム

- Webアプリのみ（スマホアプリは初期段階では不要）

### ユーザープロフィール

| 項目 | 必須/任意 | 備考 |
|------|----------|------|
| ニックネーム | 必須 | 匿名性を確保 |
| 年代 | 必須 | 20代・30代・40代…のように選択式 |
| 性別 | 必須 | マッチング時のフィルター条件に使用 |
| 好きな食べ物・飲み方の傾向 | 必須 | ガッツリ系/軽め/とにかく飲む 等 |

- 自己紹介文や写真は不要（匿名性を重視）

### マッチング方式

- **イベント投稿型** — 主催者が「この日・この店で食べ放題行きたい」と投稿し、参加者が応募する形式
- チャット機能は不要（投稿の日時・場所に集合するだけのシンプルな設計）

### イベント投稿の項目

| 項目 | 内容 |
|------|------|
| 日時 | 開催日・集合時間 |
| お店 | 主催者が決めて投稿する |
| 地域 | 都道府県・エリア（例: 東京都 新宿） |
| 食べたいもの | カテゴリ選択（大カテゴリ＋サブカテゴリ） |
| 予算感 | 一人あたりの予算帯（例: 3,000〜4,000円） |
| 募集人数 | 3〜5人程度の少人数グループ |
| 性別フィルター | 「誰でもOK」「同性のみ」「男性のみ」「女性のみ」を選択可能 |

### 検索機能

以下の条件で絞り込める。

- **予算感** — 価格帯で絞り込み
- **食べたいもの** — 大カテゴリ＋サブカテゴリで絞り込み
- **地域** — 都道府県・エリアで絞り込み
- **日時** — 日付で絞り込み
- **性別フィルター** — 自分が参加可能なイベントのみ表示

### 食べ物カテゴリ（案）

大カテゴリとサブカテゴリの2階層。

| 大カテゴリ | サブカテゴリ例 |
|-----------|-------------|
| 焼肉 | ホルモン / 韓国BBQ / 高級焼肉 |
| 寿司 | 回転寿司 / 高級寿司 |
| 中華 | 中華料理 / 火鍋 / 飲茶 |
| イタリアン | ピザ / パスタ |
| 居酒屋 | 和風居酒屋 / 創作居酒屋 / 大衆酒場 |
| しゃぶしゃぶ・すき焼き | — |
| ビュッフェ・バイキング | ホテルビュッフェ / スイーツビュッフェ |
| その他 | エスニック / 海鮮 / 鉄板焼き 等 |

※ サブカテゴリは今後ユーザーの利用傾向を見て拡充

### 支払い方法

- **現地で各自支払い**（アプリは決済に関与しない）
- 完全割り勘が前提

### ドタキャン対策

- **信頼スコア制** — ドタキャンすると評価が下がる
- 参加後に主催者・参加者の双方が「ちゃんと来たか」を報告する仕組み
- 信頼スコアが低いユーザーは検索結果で表示順位が下がる等のペナルティ

### 安全対策

- 通報・ブロック機能
- 通報が一定数に達したアカウントは利用停止

### 収益化

- 未定（今後検討）

## 今後検討が必要な項目

| 項目 | 検討内容 |
|------|---------|
| 本人確認 | 将来的に身分証提出を導入するか |
| キャンセル期限 | 何日前までキャンセル可能にするか |
| リピート機能 | 過去に一緒に参加した人をお気に入り登録できるか |
| レビュー機能 | イベント参加後に「楽しかった」等の簡易レビューを残せるか |
| お店の情報連携 | ぐるなびやホットペッパーのAPIと連携するか |
| 通知 | 条件に合うイベントが投稿されたときのメール/Push通知 |
| 収益モデル | 広告 / プレミアムプラン / 提携店舗からの手数料 等 |
